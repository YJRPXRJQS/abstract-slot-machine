# B站抽象视频生成器 - 赛博老虎机

## Concept & Vision

一个融合赛博朋克美学与玄学抽卡乐趣的B站抽象视频创作提示词生成器。用户通过拉动老虎机的交互形式，随机组合网络热梗、视觉风格、视频模板等元素，获得荒诞而又有迹可循的创作灵感。整体氛围应该是"电子菩萨在线发功"——严肃又搞笑，正经又抽象。

## Design Language

### Aesthetic Direction
赛博朋克 + 蒸汽波 + 虎口拔牙的荒诞感。霓虹灯效果、毛刺艺术(Glitch)、随机闪烁的故障美学，混合老式老虎机的机械感。

### Color Palette
- Primary: #FF2D6A (霓虹粉)
- Secondary: #00F5D4 (电光青)
- Accent: #FFE66D (警告黄)
- Background: #0D0221 (深紫黑)
- Text: #FFFFFF / #B8B8D1 (白/淡紫)
- Glow effects: rgba(255, 45, 106, 0.6)

### Typography
- 主标题: "ZCOOL KuaiLe" 或 "Noto Sans SC" 粗体
- 正文: "Noto Sans SC" Regular
- 装饰/标签: 等宽字体感

### Motion Philosophy
- 老虎机旋转: 3D透视滚动，带随机速度变化
- 停止: 依次停止，带轻微回弹效果(bounce)
- 结果展示: 闪光特效 + 文字打字机效果
- 背景: 缓慢移动的扫描线和粒子

## Features & Interactions

### 核心功能

#### 1. 六大滚轮系统
- **经典老梗组** (10-15个): 牢大、awsl、爷青回、YYDS、绝绝子、耗子尾汁、针不戳、芜湖、打工人、摆烂、带篮子、马保国、窃格瓦拉、东莞仔等
- **实时热梗组** (5-8个): 雪山狐狸、精神状态、meme热图等月度热点
- **视频形式组** (8-10个): AI视频模板、综艺模仿（最强大脑、变形记、天宫课堂）、PPT式、聊天记录式、屏录式
- **视觉风格组** (8-10个): 赛博朋克、蒸汽波、像素风、水墨风、故障艺术、醣潮风、极简主义、泼墨风
- **抽象音效组** (8-10个): 电梯音效、恐怖音效、新闻联播BGM、京剧鼓点、空耳空耳
- **配色方案组** (5-8个): 赛博粉青、老照片棕、霓虹夜市、莫兰迪灰

#### 2. 老虎机交互
- 点击"开始抽取"后，六个滚轮依次启动旋转
- 滚动2-3秒后依次停止（有轻微回弹）
- 每个滚轮停止时播放"咔哒"音效
- 全停止后触发结果展示动画

#### 3. 结果生成
- 根据组合自动生成荒诞但有逻辑的视频标题
- 输出可复制的中文提示词
- 生成3个参考意向图关键词（用于以图搜图）
- 提供音效和平台发布建议

### 交互细节
- 悬停滚轮: 轻微发光效果
- 滚动中: 禁止重复点击
- 结果区: 打字机效果展示文字

## Component Inventory

### SlotMachine 组件
- 6个独立的滚轮，3D透视滚动
- 每轮显示3个候选项（中间项为选中项）
- 滚动时有模糊效果
- 停止时有弹性动画

### ResultCard 组件
- 毛玻璃卡片效果
- 分区展示各类型结果
- 复制按钮带反馈动画

### SpinButton 组件
- 渐变背景 + 发光边框
- 点击有按压效果
- 禁用状态: 灰色 + 禁止图标

## Technical Approach

- 纯前端实现: HTML + CSS + Vanilla JavaScript
- 无需后端，所有数据存储在JS对象中
- 使用CSS 3D transform实现滚轮效果
- CSS动画实现视觉效果
- LocalStorage可选存储历史记录
