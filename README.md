# 🎰 赛博老虎机 - B站抽象视频生成器

一个结合网络热梗和AI创意的视频灵感生成工具。

## ✨ 功能特性

- 🎲 **老虎机抽取** - 随机组合经典梗、热梗、视频形式、视觉风格等元素
- 🔒 **锁定功能** - 可以锁定喜欢的元素，只随机其他部分
- 🚫 **禁用功能** - 不需要的分类可以直接禁用
- ✏️ **自定义关键词** - 添加你自己的梗到词库
- 🤖 **AI脚本生成** - 基于抽取结果，让Kimi AI生成15/30/45秒视频脚本

## 🚀 快速开始

### 方式1：使用本地代理（推荐）

1. **确保已安装Node.js**
   - 下载地址：https://nodejs.org/
   - 安装LTS版本即可

2. **启动本地服务器**
   ```bash
   # Windows
   双击 start.bat

   # 或命令行
   node proxy-server.js
   ```

3. **访问应用**
   - 打开浏览器访问：`http://localhost:3000`

4. **使用AI功能**
   - 先抽取老虎机获得创作元素
   - 选择视频时长（15s/30s/45s）
   - 点击"生成抽象脚本"
   - 等待AI生成带时间轴的完整脚本

### 方式2：直接打开（无AI功能）

直接双击 `index.html` 用浏览器打开，但AI脚本生成功能需要本地代理才能正常工作。

## 🤖 AI脚本生成

### 支持的AI模型

- **Kimi（默认）** - Moonshot AI，已配置好API Key
- **OpenAI GPT-4** - 需要自备API Key
- **Claude 3** - 需要自备API Key
- **Google Gemini** - 需要自备API Key

### 使用说明

1. 抽取老虎机获得创作元素
2. 在AI脚本生成器区域选择时长
3. 点击"生成抽象脚本"按钮
4. 等待AI生成包含以下内容的脚本：
   - 📋 脚本标题
   - ⏱️ 详细时间轴
   - 🎬 画面描述
   - 💬 台词/字幕
   - 🏷️ 推荐标签

## ⚠️ 注意事项

1. **API Key安全**
   - 默认已配置Kimi API Key
   - 如需使用其他AI模型，请自行获取API Key

2. **CORS跨域**
   - 浏览器直接调用API会遇到CORS限制
   - 使用本地代理服务器可解决此问题

3. **网络要求**
   - 需要能访问互联网（调用AI API）

## 🔧 技术栈

- 前端：HTML5 + CSS3 + Vanilla JavaScript
- 后端：Node.js (代理服务器)
- AI：Kimi API (Moonshot AI)
